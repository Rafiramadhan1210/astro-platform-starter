1.Apa itu Virtual box

VirtualBox adalah perangkat lunak virtualisasi open-source gratis dari Oracle yang memungkinkan Anda membuat dan menjalankan "mesin virtual" (VM) untuk sistem operasi yang berbeda secara bersamaan di satu komputer fisik. Ini memungkinkan pengguna untuk menjalankan beberapa sistem operasi seperti Windows, Linux, dan macOS di komputer yang sama

2.Persiapan

  • Virtual Box (Download dari situs resminya)
    • Link download virtual box
  • File ISO Ubuntu (Download di situs resmi)
    • Link Download Ubuntu
  • RAM Minimal 4GB

3.Langkah-langkah Instalasi

  1. Install VirtualBox di Windows
  2. Buat Mesin Virtual Baru
  3. Atur RAM dan storage yang di inginkan
  4. Masukan file ISO Ubuntu yang di download tadi
  5. Jalankan dan Lakukan Instalasi

4. Tips Tambahan

Gunakan fullscreen mode (ctrl kanan + ) dan install guest addictions untuk performa lebih halus
